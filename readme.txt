Start by typing  type "npm install" to install all modules or dependencies

To start the server type "npm run server" via the command line and 
you will get a console log message indicating "Server listening on port 5000"

You can test this program via the front-end ejs file in "http://localhost:5000/api/upload" or via post-man or any api testing tool.

Upon successful file upload in either the front-end ejs file or postman, you will get a message saying 
"Uploaded Successfully" in json format with the accompanying file details and the cloudinary url source.
